# Datasets

This document lists the key datasets used and referenced in this project.

## Primary Datasets

*   **Doench 2016 gRNA efficiency dataset**
    *   **Reference**: Doench et al. (2016) - Optimized sgRNA design (Nature Biotechnology)
    *   **Description**: A widely used dataset for CRISPR guide RNA efficiency.

*   **Kim 2025 gRNA dataset**
    *   **Reference**: Kim et al. (2025) - Large-scale gRNA efficiency dataset
    *   **Description**: A large-scale dataset of gRNA efficiency with 18,102 guides.

*   **BioGRID-ORCS Homo sapiens v1.1.17**
    *   **Description**: A dataset for validation.

## Other Referenced Datasets

*   **PCSK9 sequences**: 155 bp from GenBank, used as examples in `EXPERIMENTAL_VALIDATION.md`.
